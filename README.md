🕷 Spider-Man Multiverse

Projeto front-end inspirado no multiverso dos filmes do Homem-Aranha, efeitos visuais e manipulação de DOM utilizando JavaScript puro.

🚀 Sobre o Projeto

O projeto recria uma experiência imersiva baseada nas diferentes versões cinematográficas do Homem-Aranha, incluindo:

 - Página inicial com efeito dinâmico de alteração de background

 - Carrossel interativo de personagens

 - Linha do tempo animada com efeitos de hover

 - Alteração dinâmica do fundo ao passar o mouse

 - Cards com overlay animado e animações suaves

 - Páginas individuais de cada filme

 - Vídeo de fundo nas páginas dos filmes

 - Galeria interativa de imagens

🛠 Tecnologias Utilizadas

 - HTML5

 - CSS3 (Flexbox, animações, pseudo-elementos, transitions)

 - JavaScript (manipulação de DOM e eventos)

🎯 Principais Conceitos Praticados

 - Manipulação de eventos (mouseenter, mouseleave)

 - Alteração dinâmica de classes no body

 - Uso de data-attributes

 - Efeitos visuais com transform, opacity e transition

 - Organização de layout com Flexbox

 - Estruturação de projeto front-end

🌐 Estrutura do Projeto

 - Página inicial com carrossel interativo

 - Seção de linha do tempo do multiverso
